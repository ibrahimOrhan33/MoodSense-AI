Features
Biometric Security: Access control via face recognition using the Facenet512 model.
Emotion Analysis: Detects real-time emotions (Happy, Sad, Angry, etc.) and adjusts the AI's greeting tone.
Local AI Brain: Powered by Ollama (Llama 3.2), ensuring all conversations remain private and local.
Real-time Data Integration: * Financial data via yfinance (Exchange rates, etc.).
Web search capabilities via googlesearch-python.
Session Management: Logs login/logout events into a CSV file and maintains user-specific chat history.
Modern UI: A clean, dark-themed interface built with PyQt5.
